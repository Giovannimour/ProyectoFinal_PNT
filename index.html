Machine Learning

1. Supervised Learning

  • Supervised Learning

    • Algorithm learns from labeled data, then makes predictions on new, unlabeled data.

2. Unsupervised Learning

  • Algorithm learns from unlabeled data to identify hidden patterns or data groupings.

3. Reinforcement Learning

  • Algorithm learns by interacting with an environment and receives rewards for correct decisions

Artificial Neural Networks

  • A subfield of machine learning inspired by biological neural networks.

Recurrent Neural Networks

  • Used in situations where data is sequential, such as in language translation or speech recognition.
